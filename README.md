# 🎓 Lessons Rings — Расписание звонков

Настольное приложение для отображения расписания звонков и новостей школы. Разработано с нестандартным дизайном, анимациями и гибкой кастомизацией под фирменный стиль образовательного учреждения.

---

## 🚀 Возможности

- 🔵 **Нестандартный UI:** переключение через скрытые элементы, которые реагируют на наведение мыши 
- 📡 **Парсинг новостей:** автоматическая загрузка новостей с сайта школы через модуль 
- 💾 **Локальная БД:** хранение расписания и настроек в SQLite3
- ⚙️ **Гибкая кастомизация:** шрифты, отступы, видимость вкладок, громкость звука
- 🔊 **Голосовое оповещение:** озвучка перемен 
- 📁 **Экспорт/импорт настроек:** через txt-файл для переноса конфигурации
- 📐 **Адаптивность:** интерфейс подстраивается под разрешение экрана
